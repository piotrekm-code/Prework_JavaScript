## Ocena literowa

Z wykorzystaniem instrukcji warunkowej `switch` wyświetlcie odpowiedni tekst 
w zależności od wartości zmiennej `grade`:

- dla `6` tekst **"A"**
- dla `5` tekst **"B"**
- dla `4` tekst **"C"**
- dla `3` tekst **"D"**
- dla `2` tekst **"E"**
- dla `1` tekst **"F"**
